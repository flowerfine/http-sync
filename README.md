# Http-Sync

众多电商、直播、saas 企业会提供开放平台供客户使用 API 对接，支持生态解决方案。对接开放平台就成为了一个技术需求。

作者来自大数据团队，对接众多开放平台，同步企业自身数据供团队和公司数据分析使用，`http-sync` 项目即是通过 `HTTP` 对接开放平台实现数据同步的开源项目。
